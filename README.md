# Telegram Bot Deployment

## Setup Instructions

1. **Install Dependencies**  
   ```sh
   npm install
   ```

2. **Run the Bot**  
   ```sh
   npm start
   ```

3. **Deploy on Pterodactyl**  
   - Clone this repository onto your Pterodactyl server.
   - Run `npm install` to install dependencies.
   - Use `node UPDATED_CORRECTED.js` or `npm start` to launch the bot.

## Notes  
- Make sure your bot token is correctly set in `UPDATED_CORRECTED.js`.  
- Keep your token secret to prevent unauthorized access.
